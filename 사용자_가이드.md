## 1. 시작하기

### 로그인 방법

접속하면 로그인 화면이 나타납니다. **두 가지 방법**으로 로그인할 수 있습니다.

#### Google 계정으로 로그인 (권장)
1. **"Google 계정으로 계속하기"** 버튼 클릭
2. Google 계정 선택 팝업에서 **@udimpact.ai 계정** 선택
3. 로그인 완료 후 자동으로 메인 화면 이동

> ⚠️ **주의**: `@udimpact.ai` 도메인 계정만 Google 로그인이 가능합니다.
> 다른 Google 계정으로 시도하면 "회원가입이 불가능한 계정" 안내가 표시됩니다.

#### 이메일/비밀번호로 로그인
| 계정 | 비밀번호 |
|------|----------|
| zero@udimpact.ai | underdogs2024! |
| admin@underdogs.co.kr | underdogs2024! |

### 로그아웃
화면 우측 상단의 사용자 이름 옆 **로그아웃 버튼(→)** 을 클릭합니다.

## 2. 코치 검색 & 필터링

로그인 후 첫 화면이 **코치 검색 메인 페이지**입니다.

### 화면 구성

```
┌─────────────────┬────────────────────────────────────┐
│                 │  상단 툴바 (뷰 모드 / AI추천 / 등록) │
│   필터 패널     ├────────────────────────────────────┤
│   (좌측 고정)   │                                    │
│                 │       코치 카드 목록               │
│                 │                                    │
└─────────────────┴────────────────────────────────────┘
                  [하단 선택 바 — 선택 시 표시]
```

### 키워드 검색
필터 패널 상단의 검색창에 코치 **이름, 소속, 전문분야** 키워드를 입력하면 실시간으로 결과가 필터링됩니다.

### 필터 항목

| 필터 | 설명 |
|------|------|
| **티어** | 베테랑코치 / UD코치 / 외부풀 |
| **카테고리** | 파트너코치, 코치, 컨설턴트, 투자사, 글로벌코치, 로컬크리에이터, 특강연사 |
| **국적** | 국가별 필터 |
| **전문분야** | 다중 선택 가능 |
| **경험 업종** | 산업군별 필터 |
| **코칭 가능 지역** | 활동 지역별 필터 |
| **역할** | 역할 유형 필터 |
| **해외 코칭** | 해외 교육 가능 여부 (모두 / 가능 / 불가) |
| **언어** | KR / EN / JP |

- 각 필터 항목 옆의 숫자는 **해당 조건에 해당하는 코치 수**입니다.
- 필터 패널 상단의 **"초기화"** 버튼으로 모든 조건을 한 번에 지울 수 있습니다.

### 뷰 모드 전환
상단 툴바에서 두 가지 모드를 전환할 수 있습니다.

- **추천 코치**: AI 또는 필터 기준으로 정렬된 추천 결과 (추천 인원 수 설정 가능: 3/5/7/10/15/20/50명)
- **전체 결과**: 필터 조건에 맞는 모든 코치 (스크롤로 추가 로드)

## 3. AI 코치 추천

사업 내용 또는 RFP 텍스트를 입력하면 **Gemini AI가 분석**해 적합한 코치를 자동 추천합니다.

### 사용 방법

1. 필터 패널 하단 또는 상단 툴바의 **"AI 맞춤 코치 추천"** 버튼 클릭
2. 팝업창에 사업 내용 또는 필요한 코치 조건을 **자유롭게 입력**
   - 예시 버튼을 클릭하면 샘플 텍스트가 자동 입력됩니다
3. **"추천받기"** 버튼 클릭 후 분석 완료 대기 (5~15초 소요)
4. 결과 화면에서 추천 코치 목록 확인 — 상단에 **분석 요약 배너** 표시

### 입력 팁

> 아래 정보를 포함할수록 추천 정확도가 높아집니다:
> - **사업명 및 주관 기관** (예: 중기부 K-스타트업 2026)
> - **필요 코치 수** (예: 코치 20명 필요)
> - **전문 분야** (예: AI/딥테크, 제조, ESG)
> - **활동 지역** (예: 서울·수도권)
> - **역할** (예: 창업 코칭, 멘토링, 강의)

### AI 분석 결과 배너
추천 완료 후 코치 목록 위에 배너가 표시됩니다:
- 사업명, 요약 설명
- 필요 도메인 태그 / 핵심 역량 태그
- 인식된 코치 수 (예: `코치 30명`)
- 배너 우측 **✕** 버튼으로 닫으면 일반 검색으로 복귀

### 필터와 함께 사용하기
AI 추천 전에 필터(티어, 지역, 역할 등)를 설정하면, **필터 범위 내에서만** 추천 결과가 표시됩니다.

> AI 서버 응답이 없을 경우 자동으로 키워드 기반 매칭으로 전환됩니다.

## 4. 코치 상세 정보 보기

코치 카드를 클릭하면 **상세 정보 모달**이 열립니다.

### 확인 가능한 정보

- 이름, 국적, 소속, 직책, 지역
- 이메일 / 전화번호
- 학력 / 경력
- **소개** — 코치 소개글
- **경력 이력** — 주요 경력 내역
- **현재 하고 있는 일** — 현재 활동 내용
- **언더독스와의 협력 이력** — 과거 협업 사업 이력

## 5. 코치 선택 & 자료 내보내기

코치를 선택해 제안서용 Word 문서 또는 PowerPoint 파일을 자동 생성할 수 있습니다.

### 코치 선택하기

- 코치 카드 **우측 상단의 체크박스** 클릭 → 선택됨 (카드 좌측에 빨간 인디케이터 표시)
- 상단 툴바의 **"전체 선택"** 버튼으로 현재 화면의 코치 전체 선택 가능
- 선택된 코치는 **화면 하단의 선택 바**에 미니 아바타로 표시됨
- 아바타 옆 **×** 버튼으로 개별 해제, **"모두 지우기"**로 전체 해제

### 자료 내보내기

1. 하단 선택 바의 **"내보내기"** 버튼 클릭
2. 드롭다운에서 형식 선택:
   - **Word (DOCX)** — 코치 프로필 문서
   - **PowerPoint (PPTX)** — 발표용 슬라이드
3. **프로젝트명** 입력 (파일명에 반영됨)
4. 파일 자동 다운로드

## 6. 사업 관리

코치 투입 현황, 단가, 평가를 사업 단위로 관리할 수 있습니다.

### 사업 관리 페이지 이동
상단 툴바의 **"사업 관리"** 버튼 클릭

---

### 6-1. 사업 등록

1. **"+ 새 사업"** 버튼 클릭
2. 다음 정보 입력:

| 항목 | 필수 | 설명 |
|------|------|------|
| 사업명 | ✅ | 사업 이름 |
| 발주처 | | 클라이언트 기관명 |
| 사업 설명 | | 간략한 사업 내용 |
| 시작일 / 종료일 | | 사업 기간 |
| 상태 | ✅ | 기획 중 / 진행 중 / 완료 |
| 총 사업 예산 | | 전체 예산 (원) |

3. **"저장"** 클릭

---

### 6-2. 코치 투입

1. 사업 카드 클릭 → 사업 상세 화면 진입
2. **"코치 추가"** 버튼 클릭
3. 코치 검색 후 선택 → 투입 완료

---

### 6-3. 단가 & 지급액 입력

사업 상세 화면의 코치 카드에서 입력합니다.

| 항목 | 설명 |
|------|------|
| **회차(일)당 단가** | 1회 진행 시 단가 (원) |
| **투입 횟수/일수** | 전체 진행 횟수 |
| **총 지급액** | 단가 × 횟수로 자동 계산, 직접 수정도 가능 |

- 단가와 횟수를 입력하면 총 지급액이 **자동 계산**됩니다.
- 총 지급액을 직접 입력해 **수동으로 덮어쓸** 수도 있습니다.
- 사업 예산을 입력했다면 **예산 대비 지출 비율(%)** 이 자동 표시됩니다.

---

### 6-4. 과업 내용 기록

코치 카드의 **"과업 내용"** 텍스트창에 해당 코치가 수행한 업무를 입력하고 저장합니다.

---

### 6-5. 코치 평가

1. 코치 카드의 **"평가 작성"** 버튼 클릭
2. **별점 (1~5점)** 선택
3. **코멘트** 작성
4. **"저장"** 클릭

- 평가 완료 후에는 **"평가 수정"** 버튼으로 재작성 가능합니다.
- 사업 카드에는 **"평가 X/Y건 완료"** 진행도가 표시됩니다.

---

### 6-6. 사업 수정 & 삭제

- 사업 카드 우측 상단의 **연필(✏) 아이콘** → 사업 정보 수정
- 사업 상세 화면 하단의 **"삭제"** 버튼 → 사업 삭제 (복구 불가)

## 7. 코치 현황 대시보드

사업 관리 페이지 상단 **"코치 현황"** 탭에서 확인합니다.

### 확인 가능한 정보

**요약 카드 (상단)**
- 전체 투입 코치 수
- 전체 총 지급액 합계
- 전체 평균 평점

**코치별 집계 테이블**

| 컬럼 | 설명 |
|------|------|
| 코치명 | 이름 및 티어 |
| 참여 사업 수 | 투입된 사업 건수 |
| 총 투입 횟수 | 전체 사업 합산 |
| 총 지급액 | 전체 사업 합산 지급액 |
| 평균 평점 | 별점 평균 (평가 완료 건 기준) |
| 참여 사업 | 사업명 목록 |

**정렬 옵션**: 지급액순 / 평점순 / 투입 횟수순
**검색**: 코치명으로 검색 가능

## 8. 코치 정보 직접 등록 & 수정

기존 DB에 없는 코치를 직접 추가하거나 기존 정보를 수정할 수 있습니다.

> 등록/수정한 정보는 **Firebase Firestore에 저장**되어 모든 팀원의 브라우저에 **실시간으로 자동 반영**됩니다.

### 신규 등록
1. 상단 툴바의 **"+ 신규 등록"** 버튼 클릭
2. 코치 정보 입력 (이름 필수)
3. **"저장"** 클릭 → 코치 목록에 즉시 반영

### 기존 코치 수정
1. 수정할 코치 카드에 **마우스를 올리면** 편집 버튼 표시 (관리자 계정만)
2. 편집 버튼 클릭 → 수정 폼 열기
3. 정보 수정 후 **"저장"**

### 삭제
수정 폼 하단의 **"삭제"** 버튼 클릭 (직접 추가한 코치만 삭제 가능)

## 9. 자주 묻는 질문

**Q. Google 로그인 시 "회원가입이 불가능한 계정" 오류가 뜹니다.**
A. `@udimpact.ai` 도메인 계정만 Google 로그인이 가능합니다. 팝업에서 올바른 계정을 선택해 주세요.

**Q. AI 추천이 작동하지 않습니다.**
A. AI 서버 응답이 없을 경우 자동으로 키워드 기반 매칭으로 전환됩니다. 결과는 정상 표시됩니다.

**Q. AI 추천에서 코치를 30명 요청했는데 30명이 안 나옵니다.**
A. AI가 RFP에서 코치 수를 인식하면 해당 인원을 우선 추천합니다. 단, 현재 설정된 필터 조건에 맞는 코치가 30명 미만일 경우 필터 범위 내에서 가능한 최대 인원을 추천합니다.

**Q. 내가 등록한 코치가 다른 팀원에게도 보이나요?**
A. 네, 직접 등록하거나 수정한 코치 정보는 Firebase Firestore에 저장되어 모든 팀원에게 실시간으로 공유됩니다. 저장 즉시 다른 브라우저에도 자동 반영됩니다.

**Q. 사업 데이터는 어디에 저장되나요?**
A. 사업 정보(투입 코치, 단가, 평가 등)는 Firebase Firestore에 저장됩니다. 로그인 계정 기준으로 공유되며, 브라우저를 닫아도 유지됩니다.

**Q. Word/PPT 내보내기 파일에 포함되는 내용은 무엇인가요?**
A. 선택한 코치들의 프로필 정보(이름, 소속, 전문분야, 소개, 경력 등)가 포함됩니다.

**Q. 로그인 후 세션이 얼마나 유지되나요?**
A. 브라우저 탭을 닫으면 세션이 종료됩니다. 재접속 시 다시 로그인이 필요합니다.
